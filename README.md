AlphaViewer is a crossplatform application for view and use wallpaper from alphacoders.com.

## Build:
```bash
git clone https://github.com/Wickinats/AlphaViewer.git
cd AlphaViewer
npm install

# It is not all. Now just run "index.html" in a browser.
```
